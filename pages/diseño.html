Aprende Diseño Web. Tutoriales y Recursos para Crear Sitios Web Impresionantes

Maestría en CSS: Domina el estilo y la apariencia de tus páginas web

En este video, te enseñaremos todo lo que necesitas saber para convertirte en un maestro de CSS y llevar la apariencia de tus páginas web al siguiente nivel. Descubre cómo utilizar las últimas técnicas y herramientas para crear diseños increíbles, desde la selección de colores y fuentes hasta la organización y optimización de tu código CSS. Con este curso, estarás en el camino hacia el éxito como diseñador web y serás capaz de crear sitios web visualmente atractivos y profesionales que se destacarán en la web

Aprende a crear un diseño web responsive desde cero con Bootstrap

En este video tutorial de Bootstrap 5, descubrirás cómo crear un diseño web responsive desde cero utilizando las herramientas de Bootstrap. Aprenderás sobre los breakpoints, containers, grid y columns, y cómo utilizarlos para crear un diseño que se adapte a diferentes tamaños de pantalla. Ya seas principiante o tengas experiencia en diseño web, este tutorial es una excelente manera de mejorar tus habilidades y crear diseños web modernos y atractivos.

Herramientas de Prototipado para Validar tus Diseños Web y Móvil

Descubre 4 herramientas esenciales para crear prototipos de sitios web y aplicaciones móviles o de escritorio, y así validar tus diseños con el cliente. Es crucial asegurarse de que tanto el diseño como la experiencia de usuario sean adecuados antes de empezar a programar. Aprenderás sobre herramientas que van desde las más simples, como Pop by Marvel, hasta las más avanzadas, como Principle, que ofrecen una experiencia de usuario muy realista con interacción. También verás cómo Keynote (o incluso PowerPoint) e InVision pueden ayudarte en el proceso de prototipado.
